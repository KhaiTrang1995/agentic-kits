# Zabbix Severity Guide

| Zabbix | Meaning | Typical response |
|--------|---------|------------------|
| Not classified | Info / inventory | No page |
| Information | FYI | Backlog |
| Warning | Review in shift | Team channel |
| Average | Partial impact | On-call note |
| High | Core degradation | Page on-call |
| Disaster | Outage / data risk | War room |

When unsure between two levels, pick the **higher** one and demote after evidence.
