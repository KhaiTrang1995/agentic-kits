🎓 Tự tạo game trong 10 phút

npm run dev
npm run new-game skate -- --template wing --title "SKATE"
  1. Chọn template gần với kiểu điều khiển bạn muốn.
  2. Sửa CONFIG, mesh nhân vật, và place() chướng ngại.
  3. Đổi CSS variables (--accent, --bg) cho theme riêng.

Đọc chi tiết: docs/LEARN.md · kiến trúc: README.md
